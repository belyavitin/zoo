Тестовое задание.

3д зоопарк, в котором можно увидеть различных животных, с возможностью расширения видов и поведения животных до тысяч.

Так как ограничения по реализации не было, решил попробовать сделать максимально быстро, по минимуму используя нестандартные и необычные подходы или фреймворки, поэтому делал на компонентной модели по стандартам юнити.
Использовал простенькую систему событий для передачи данных в UI и немножко поигрался с визуалом и поведением животных чтобы было приятнее смотреть.
